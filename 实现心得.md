## 3. 访问接口

update 函数绑定了三个链接
```
update: function (message) {
      this.url = message
      this.service_call()
    },

```
service_call函数 绑定搜索按钮

```
    service_call: function() {
      var that = this;
      //that.sites = "fuck"

  

      that.axios.get("https://swapi.co/api/" + that.url + "?format=json")
      .then(function(res) {
        console.log(res.data)
        that.sites = res.data

      })


```
